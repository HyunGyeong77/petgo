# PetGo

<br>

## 시작하게 된 계기

강아지 관련 영상을 보며 힐링을 느끼는 과정에서,   
실제 보호자에게 도움이 되는 정보는   
여러 사이트에 흩어져 있다는 점을 느꼈습니다.   
강아지를 처음 키우는 보호자가   
기본적인 정보를 한 곳에서 확인할 수 있으면 좋겠다고 생각해   
이 사이트를 기획하게 되었습니다.

<br>

## 기술 스택

- ⚛️ **React / Next.js** - UI 및 서버사이드 렌더링
- 🎨 **SCSS** - 스타일링
- 📝 **TypeScript** - 타입 안정성
- 🔧 **Git** - 버전 관리
- 🖌️ **Figma** - 디자인
