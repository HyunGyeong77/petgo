# Landing Page

## Recommend Section

### 목적
- Carousel 기능 추가
- 아이템 수가 늘어날 때 캐러셀이 계속 동작하도록 개선

### 구성
- 기본 화면에서 동시에 보이는 아이템: 5개
- 화면에 보이는 아이템 수에 따라 한 번에 이동할 아이템 수 조정
  - 동시에 보이는 아이템 수 >= 화면에 표시되는 아이템 수 → 한 번에 동시에 보이는 아이템 수만큼 이동
  - 그 외 → 한 번에 1개씩 이동
- 아이템 간 간격: 1rem

### 요구사항
- 반응형
  - 모바일: 1개
  - 태블릿: 3개
  - 데스크탑: 5개
- 파일 작업: `app/_components/recommend/category-block/CategoryBlock.tsx`
- 이전/다음 버튼
  - 이전 상품이 없을 시 이전 버튼 비활성화
  - 다음 상품이 없을 시 다음 버튼 비활성화
  - 클릭 시 화면에 보이는 아이템 수 기준으로 이동
